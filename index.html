Text input fields

Single line text fields type = text


  • type: Сообщает браузеру, к какому типу относится элемент формы (possible value: email, password, search, phone number etc).
  • value: Дефолтное значение элемента
  • name : Имя поля, предназначено для того, чтобы обработчик формы мог его идентифицировать.
  • minlength/maxlength=8: (the maximum number of characters that can be entered into the box).

Multi-line text fields textarea

  • cols: The visible width of the text control, in average character widths.
  • rows: The number of visible text lines for the control.
